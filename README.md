**CricPulse – Project Description**

**Overview:**
CricPulse is an interactive cricket match analysis web application designed to give users a real-time understanding of match situations, especially during run chases and innings progression. It allows users to simulate or analyze live match scenarios by selecting teams, formats, and match conditions. The platform combines clean UI design with powerful calculations to present insights like run rates, projections, win probabilities, and match pressure in a visually engaging way.

**Objective:**
The main objective of CricPulse is to simplify cricket analytics for fans and students by turning complex match data into easy-to-understand insights. It aims to help users quickly evaluate match situations, predict outcomes, and understand the dynamics of scoring, chasing, and performance in different formats like T20, ODI, and custom matches.

**Key Features:**

* **Team Selection System:** Users can search and select teams from international and league cricket (IPL, BBL, etc.).
* **Innings-Based Analysis:** Separate modes for 1st innings (score projection) and 2nd innings (chase calculation).
* **Dynamic Match Inputs:** Users can input runs, overs, wickets, and targets to simulate real scenarios.
* **Run Rate Calculations:** Automatically computes Current Run Rate (CRR) and Required Run Rate (RRR).
* **Win Probability Estimation:** Provides a visual percentage-based prediction of which team is likely to win.
* **Score Projection Engine:** Predicts final scores based on current performance and match conditions.
* **Visual Insights:** Includes progress bars, match status indicators, and overs visualization for better understanding.
* **Live Match Finder (UI-based):** Allows users to explore or simulate live match scenarios.
* **Custom Match Formats:** Supports T20, ODI, The Hundred, and user-defined formats.

CricPulse blends cricket knowledge with interactive technology, making match analysis fast, intuitive, and engaging.
